Construct observable specialized with passed callback function. Most easiesest way to construct observable "{}on the fly"{} via lambda and etc. 



\begin{DoxyWarning}{Warning}
Be sure, that your callback doesn\textquotesingle{}t violates observable rules\+: 1) observable must to emit emissions in serial way 2) observable must not to call any callbacks after termination events -\/ on\+\_\+error/on\+\_\+completed 

Keep in mind, obtained observer is non-\/copyable, but movable by default. So, prefer perfect-\/forwarding. In case of you need to copy observer, cast it it \doxylink{classrpp_1_1dynamic__observer}{dynamic\+\_\+observer} via passing it as argument type or via as\+\_\+dynamic() member function
\end{DoxyWarning}
