Detailed Description

template<rpp::constraint::decayed_type TDisposable>
class rpp::disposable_wrapper_impl< TDisposable >

Wrapper to keep disposable. Any disposable have to be created right from this wrapper with help of make function.

Member functions is safe to call even if internal disposable is gone. Also it provides access to "raw" shared_ptr and it can be nullptr in case of disposable empty/ptr gone.

Can keep weak_ptr in case of not owning disposable
