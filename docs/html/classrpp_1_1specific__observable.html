Detailed Description

template<constraint::decayed_type Type, constraint::on_subscribe_fn< Type > OnSubscribeFn>
class rpp::specific_observable< Type, OnSubscribeFn >

observable specified with specific type of OnSubscribeFn. Used to store OnSubscribeFn function as is on stack (instead of allocating it somewhere).

It has better performance comparing to rpp::dynamic_observable. Use it if possible. But it has worse usability due to OnSubscribeFn template parameter.

Template Parameters
Typeis type of value provided by this observable
OnSubscribeFnis type of function/functor/callable used during subscription on this observable
